# bensch98.github.io

Personal blog built with [Hugo](https://gohugo.io/) and hosted on [GitHub Pages](https://pages.github.com/).

## Local Development

```bash
# install hugo
go install github.com/gohugoio/hugo@latest

# development
hugo server --buildDrafts
```

## Create a New Post

```bash
hugo new content posts/my-post.md
```
